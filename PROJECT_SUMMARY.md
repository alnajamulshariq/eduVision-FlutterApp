# Project Summary

## Project Name

EduVision - Smart University Attendance & Monitoring System

## Problem Statement

Universities often manage attendance, gate movement, student communication, and admin records through separate manual processes. This can make reporting slow, reduce transparency, and make it difficult to monitor student activity in real time.

## Proposed Solution

EduVision proposes a digital university monitoring system where students, teachers, and admins use role-based dashboards. The app demonstrates smart attendance, gate entry and exit monitoring, anonymous feedback, reporting, and admin management through a premium Flutter interface.

## Main Modules

- Smart Attendance.
- Gate Entry and Exit Monitoring.
- Anonymous Messaging.
- Attendance Reports.
- User Management.
- Academic Management.

## User Roles

- Student: Views QR, attendance, gate history, and sends anonymous feedback.
- Teacher: Starts attendance, scans QR preview, reads anonymous feedback, and monitors campus presence.
- Admin: Manages users, academics, reports, gate logs, and reported anonymous messages.

## Demo Features

- Dynamic QR preview for students.
- Teacher attendance session demo.
- Course-wise attendance reports.
- Gate entry and exit timeline.
- Parent notification preview.
- Anonymous student feedback flow.
- Admin review and sender reveal preview for reported messages.
- Mock user and academic management screens.
